<h1>💁‍♂️ 투데이 링크 서비스 소개</h1>
무료로 참여할 수 있는 다양한 이벤트 정보를 한 곳에 모아놓은 웹 서비스입니다.

사용자들은 **투데이 링크** 서비스를 통해 다양한 이벤트 정보를 확인하고 링크 접속을 통해 이벤트에 참여할 수 있습니다.
